# Purpose

This document defines how SYNTERRA multilingual documentation is synchronized.

The goal is to:
- prevent architectural fragmentation;
- maintain semantic consistency;
- preserve philosophical meaning;
- synchronize LV and EN documentation.

---

# Primary Language Principle

```text
LV = PRIMARY SOURCE
```

Latvian is the primary source language because core ideas and philosophical concepts originate in Latvian.

---

# English Mirror Principle

```text
EN = GLOBAL MIRROR
```

English documentation is not an independent branch.

It is:
- synchronized;
- semantically aligned;
- structurally mirrored.

---

# Version Pairs

Each document should exist as synchronized language layers.

Example:

```text
*.lv.md
*.en.md
```

Both documents should preserve:
- same semantic object;
- same structure;
- same meaning;
- sync state.

---

# Sync Status System

Possible states:

```text
SYNCHRONIZED
EN UPDATE REQUIRED
LV UPDATE REQUIRED
SEMANTIC REVIEW REQUIRED
```

---

# Semantic Synchronization

Translation in SYNTERRA is semantic, not literal.

Important concepts must preserve:
- meaning;
- philosophy;
- structural logic.

---

# Terminology Core


Purpose:
- stabilize translations;
- preserve conceptual consistency;
- prevent terminology drift.
