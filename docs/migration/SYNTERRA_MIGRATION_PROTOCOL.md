# 1. Purpose

This protocol defines how documentation is migrated from the previous repositories into the unified SYNTERRA repository.

Source repositories:

```text
Dzintars12/SYNTERRA-LV
Dzintars12/SYNTERRA-EN
```

Target repository:

```text
Dzintars12/SYNTERRA
```

---

# 2. Core Migration Rule

```text
One idea
One semantic ID
Two language layers
No duplicated project realities
```

---

# 3. Language Rule

Every core document must eventually exist as:

```text
*.lv.md
*.en.md
```

If a document from SYNTERRA-LV is written in English, it becomes the EN layer and receives a Latvian translation.

If a document from SYNTERRA-EN contains concepts missing in LV, it is translated and integrated into the LV layer.

---

# 4. Semantic ID Rule

Documents are migrated as semantic objects.

Example:

```text
SYN-CIV-001_CIVILIZATION_MODEL.lv.md
SYN-CIV-001_CIVILIZATION_MODEL.en.md
```

Both files describe the same semantic object.
